Stupid project for counting canyes, just for fun. CIFP Francesc de Borja Moll internal meme.

## Set up 

### Requirements

+ docker && docker-compose installed

### Commands

*Start dev compose*

```bash
docker-compose build
docker-compose -f dev.docker-compose.yml
```
